The PayNext Web SDK enables you to seamlessly integrate modern payment capabilities into your web applications including subscriptions, one-time payments, and digital wallets by using a single, secure integration layer.

## Features

- **Unified Integration**: One SDK for cards, bank transfers, and wallets.
- **Secure by Default**: PCI-compliant, with built-in fraud prevention.
- **Developer Friendly**: Works out-of-the-box with React, or via direct JS integration.
- **Dynamic UX**: Support for hosted fields, localization, and custom UI embedding.

## Browser Compatibility

The SDK supports recent versions of all major browsers:

- Chrome (v100+)
- Safari (v14+)
- Firefox (v110+)
- Edge (v105+)
- Android WebView / WKWebView (for in-app flows)

<Warning>Apple Pay is supported only on Safari (iOS/macOS). Google Pay may have limited support in Android WebViews.</Warning>

## Getting Started

To get started with the PayNext SDK, follow these steps:

1. Install the SDK for your platform
2. Retrieve your client token
3. Implement the payment flow in your application

For detailed instructions, see the platform-specific guides in the sidebar.

## Requirements

- A PayNext account
- Client token for your account
- Development environment appropriate for your platform
